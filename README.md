# SPA VIVIAN — Vite + React

Instrucciones rápidas para ejecutar el proyecto localmente.

Instalar dependencias:

```bash
npm install
```

Instalar Tailwind y generar configuración (si hace falta actualizar):

```bash
npx tailwindcss init -p
```

Iniciar servidor de desarrollo:

```bash
npm run dev
```

Build para producción:

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
